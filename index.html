Ο Χρυσός Κανόνας · The Golden Rule

Η ποιότητα του prompt
= η ποιότητα της απάντησης

Garbage in, garbage out. Αν δώσεις αόριστη ερώτηση, θα πάρεις αόριστη απάντηση.
Your prompt quality determines the output quality.

Δομή Prompt · Prompt Structure

Τα 4 στοιχεία ενός καλού prompt

1. Ρόλος · Role

"You are an experienced university administrator..."
Δώσε ρόλο στην AI

2. Πλαίσιο · Context

"I work at the finance directorate of a public university in Cyprus..."
Give background information

3. Εργασία · Task

"Draft a memo to department heads about the new procurement policy..."
Be specific about what you want

4. Μορφή · Format

"In formal tone, bullet points, max 300 words, in Greek..."
Specify the output format

Παράδειγμα · Example

Κακό vs Καλό Prompt

Κακό Prompt

"Γράψε μου ένα email"

Αόριστο. Δεν ξέρει σε ποιον, για τι, τι ύφος, τι γλώσσα, τι μήκος.

Καλό Prompt

"You are a university administrator. Draft a polite but firm reminder email to department heads about the deadline for submitting budget proposals. Tone: professional, concise. Max 200 words. Include a clear call to action with the deadline date of April 15."

Ρόλος + Πλαίσιο + Εργασία + Μορφή = Εξαιρετικό αποτέλεσμα.

Τεχνικές · Techniques

Pro Tips για καλύτερα αποτελέσματα

  • Iterative refinement: Μην περιμένεις τέλειο αποτέλεσμα — ζήτα αλλαγές "Make it shorter" "Add examples"
  • Chain of thought: "Σκέψου βήμα-βήμα" — βελτιώνει πολύπλοκα tasks "Think step by step"
  • Few-shot examples: Δώσε 1-2 παραδείγματα του τι θέλεις Show, don't just tell
  • Constraints: "Μέγιστο 200 λέξεις" / "Χρησιμοποίησε bullet points" Set boundaries
  • Persona swap: "Εξήγησε σε 10χρονο" vs "Εξήγησε σε PhD φοιτητή" Adjust audience
  • Ζήτα κριτική: "Τι λάθη μπορεί να έχει αυτό;" "What might be wrong with this?"

Pro Tips · Από την Εμπειρία μου

3 πράγματα που κάνουν τεράστια διαφορά

1. Κρατήστε τη γλώσσα στοχευμένη

Μην γράφετε στα ελληνικά περιμένοντας ελληνικό output. Χτίστε το prompt στα αγγλικά (η AI δουλεύει καλύτερα), και μετά σε νέο chat μεταφράστε το αποτέλεσμα — ή ζητήστε να σας φτιάξει prompt για μετάφραση.
Build in English, translate separately

2. Ένα θέμα ανά chat

Αν αλλάξετε θέμα στο ίδιο chat, η ποιότητα πέφτει δραματικά. Ξεκινήστε νέα συνομιλία για κάθε ξεχωριστό task. Κάθε chat = ένα project.
Diverging topics = worse outputs

3. Λιγότερα tokens = καλύτερα

Η AI έχει περιορισμένο "παράθυρο" μνήμης (context window). Αν γεμίσει με άχρηστο κείμενο, χάνει focus. Κρατήστε τα prompts σας συνοπτικά & ουσιαστικά.
Token efficiency = output quality

From "says things" to "does things"

Παραδείγματα Agents

Τι μπορεί να κάνει ένα AI Agent;

  • ChatGPT Agent Mode: Ψάχνει στο web, κάνει κλικ, πληκτρολογεί — ολοκληρωμένες εργασίες
  • Manus AI: Αυτόνομη έρευνα, ανάλυση δεδομένων, δημιουργία website — viral το 2025, εξαγοράστηκε από Meta ($2B)
  • Zapier Agents: Περιέγραψε workflow σε απλά αγγλικά → αυτοματοποιεί σύνδεση apps
  • Claude Code: Δημιουργεί κώδικα, τρέχει commands, διαχειρίζεται projects (φτιάχτηκε αυτή η παρουσίαση!)

Ασφάλεια · Safety

Τι να προσέχετε

  • Μην εμπιστεύεστε τυφλά: Πάντα ελέγχετε facts, αριθμούς, citations
  • Προσωπικά δεδομένα: Μην βάζετε ονόματα υπαλλήλων, μισθούς, ευαίσθητα στοιχεία
  • Copyright: AI output μπορεί να περιέχει copyrighted υλικό
  • Bias: Τα μοντέλα κληρονομούν bias — ελέγχετε για ισορροπία
  • Διαφάνεια: Να αναφέρετε σημαντική χρήση AI στους συναδέλφους & προϊσταμένους
  • Data retention: Ελέγξτε τις ρυθμίσεις — opt-out από training αν χρειάζεται
